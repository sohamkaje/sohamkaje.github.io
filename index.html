Project 001
CyBench Optimization
LLM Systems Engineering · Prompt Engineering · Oct – Dec 2025
Overview

CyBench is a cybersecurity benchmark designed to evaluate how well large language models reason about real-world exploitation scenarios and CTF-style tasks. I improved the efficiency and accuracy of this system through three distinct engineering efforts.

What I Built
  • Chain Compression: Filtered trivial agent steps and delegated them to lightweight models, reducing total AI token usage by approximately 47% without sacrificing task quality.
  • Dynamic Memory: Implemented a persistent, JSON-backed memory system that injects cross-run execution context directly into LLM prompts, enabling the model to learn from prior attempts.
  • Feedback Loop: Logged failed actions as counterexamples and fed them back into subsequent runs, improving subtask completion rates and iteration efficiency.
Technologies
